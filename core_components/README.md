# Core Components

This directory contains full Terraform templates that are designed to be deployed as complete infrastructure solutions.

The initial available component is:

- [standard_vpc](standard_vpc/README.md): A standard AWS VPC template with public/private subnets, internet gateway, NAT gateway, route tables, and route associations.

## Purpose

Core components are intended for use when you need a reusable, end-to-end Terraform architecture pattern rather than a single configuration snippet.
