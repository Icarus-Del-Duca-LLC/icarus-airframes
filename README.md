# Icarus Airframes

This repository contains reusable Terraform artifacts for building AWS infrastructure.

The repository is organized into two main areas:

- `core_components/`: Full Terraform templates that provide complete, deployable infrastructure patterns.
- `snippets/`: Smaller Terraform code examples and reusable fragments for reference or composition.

## Available Documentation

- [Core Components](core_components/README.md)
- [Snippets](snippets/README.md)

## Notes

This root README is intended to provide a general overview and pointer structure. The lower-level directories contain more specific documentation for each template.
